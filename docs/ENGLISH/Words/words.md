# 词汇量

[墨墨背单词](https://www.maimemo.com/)
[不背单词](https://bbdc.cn/)

## 二者比较

### 墨墨背单词的优点：
- 提供多种词汇学习方式，如背单词、听写、打卡等。
- 能够根据用户的学习情况和进度智能推荐单词。
- 有一个强大的生词本管理功能，可以方便地查看和复习之前学过的单词。
- 支持多种语言的翻译功能，方便用户了解单词的含义。

### 墨墨背单词的缺点：
- 需要付费才能使用一些高级功能，如无限制的生词本、离线下载等。
- 有些用户反映词汇资源相对较少。

### 不背单词的优点：
- 提供了大量的精选词汇和例句，对英语初学者来说很实用。
- 提供拼写功能
- 支持离线学习，无需联网即可使用。
- 界面简洁，易于操作。

### 不背单词的缺点：
- 缺乏个性化的学习进度和推荐功能。
- 生词本管理功能相对简单，不如墨墨背单词的强大。
